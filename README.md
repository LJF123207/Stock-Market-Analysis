
# 行业级股市多模态分析与多智能体交易系统

## 项目简介
本项目旨在充分挖掘新闻与行业数据在股市分析中的价值，通过两大核心模块实现智能化投资研究与决策支持：

1. **多模态趋势预测与回测验证**  
   基于 Qlib 构建多种时序预测模型，融合行业新闻和时序数据，实现行业级趋势预测与策略回测。通过对比不同模型结构，筛选最优方案，为投资决策提供科学依据。  

2. **短期涨跌分析及交易建议**  
   利用多智能体框架对市场进行深度分析，结合新闻面与技术面研究结果生成交易建议，实现更加精准的短期市场判断。

---

## 功能
本项目提供三类核心功能，支持从数据探索到交易策略生成的全流程分析：

1. **股票列表**  
   - 支持按行业代码或行业名称搜索股票  
   - 展示近期技术指标及趋势图  
   - 可查看最近 20 条相关新闻，快速捕捉市场信息  

2. **趋势预测 & 策略回测**  
   - 使用 2018-2025 年 A 股数据集（包含 100 万条新闻和行业时序数据）  
   - 通过特征相加、特征拼接、交叉注意力机制及 Decoder 构建文本与时序特征的多模态融合  
   - 对比 7 种时序模型，基于行业相对收益选择最优结构  
   - 2024.02-2025.06 回测结果显示，多模态策略年化收益较单模态基线提升约 200%  
   - 用户可选择基础时序模型与融合机制，并查看收益率预测曲线及回测效果  

3. **智能分析**  
   - 基于 LangGraph 搭建多智能体框架  
   - 配置技术面研究员、新闻面研究员进行市场情况分析  
   - 配置看涨及看跌辩手进行涨跌辩论  
   - 配置交易研究员结合以上分析生成交易建议，实现从信息分析到交易决策的闭环

## 配置方法
以下提供项目环境与依赖配置的框架，供用户参考：


2. **环境准备**  
   - 使用 `conda` 或 `pip` 安装项目依赖  
   - 示例：  
     ```bash
     conda create -n stock_env python=3.10
     conda activate stock_env
     pip install -r requirements.txt
     ```

3. **项目准备**  
   - 下载 A 股历史数据及新闻数据, 数据位置放于 
      /Stock-Market-Analysis/qlib/（data_news/data_provider_bin/data_stock）
   - 配置路径及api  
      在app.py中更改数据路径
      在/Stock-Market-Analysis/industry_agents/llm_client.py中配置api_key及base_url

4. **启动**  
   - `python app.py`
   - `http://localhost:5000/`  

5. **运行项目**  
   - 启动多模态趋势预测  
   - 运行多智能体分析模块  
   - 查看预测结果与回测报告
